Cloud and local, one config.

gateway

Provider-agnostic routing engine. Trait-based adapters, named fallback chains, per-endpoint circuit breaker, budget filtering and request tracing — with a store trait for persistence.

gateway-embedded

In-process inference adapters and an on-disk model registry. The same InferenceAdapter trait as the cloud adapters, so local and cloud models compose in one routing config.
